Getting Started

1. Installation

npm install rn-expo-core

Install peer dependencies: 

npx expo install react-native-paper react-native-safe-area-context react-native-screens

2. Setup Providers

Wrap your app with AppProviders to enable all features: 

import { AppProviders } from 'rn-expo-core';

export default function App() {
  return (
    
      {/* Your app content */}
    
  );
}

3. Use the Utilities

Start using responsive components and utilities:

import { 
  ResponsiveContainer, 
  ResponsiveText, 
  useLayout, 
  useTheme 
} from 'rn-expo-core';

function MyScreen() {
  const { isDesktop } = useLayout();
  const theme = useTheme();

  return (
    
      
        Hello World
      
    
  );
}

üì± Responsive Layout

Track device size and adapt your UI:

import { useLayout } from 'rn-expo-core';

function MyComponent() {
  const { 
    width, 
    height, 
    isMobile, 
    isTablet, 
    isDesktop,
    breakpoint 
  } = useLayout();

  return (
    
      Device: {isMobile ? 'Mobile' : isTablet ? 'Tablet' : 'Desktop'}
      Breakpoint: {breakpoint}
      Size: {width}x{height}
    
  );
}
